# v1.4.0 规划：媒体发送功能

## 背景

企业微信智能机器人支持发送图片、语音、视频、文件等媒体类型。当前 wecom-aibot-mcp 只实现了文本/markdown 和模板卡片消息，需要扩展媒体发送能力。

---

## 功能需求

### 支持的媒体类型

| 类型 | 格式 | 大小限制 | SDK 方法 |
|------|------|---------|---------|
| 图片 | png/jpg/gif | ≤ 10MB | `sendMediaMessage` |
| 语音 | amr | ≤ 2MB | `sendMediaMessage` |
| 视频 | mp4 | ≤ 10MB | `sendMediaMessage` |
| 文件 | 任意 | ≤ 20MB | `sendMediaMessage` |

## 技术要点

### 分片上传流程

SDK 已封装三步上传：
1. `aibot_upload_media_init` - 初始化，获取 upload_id
2. `aibot_upload_media_chunk` - 分片上传（每片 ≤ 512KB）
3. `aibot_upload_media_finish` - 完成，获取 media_id

### media_id 有效期

- 上传后 3 天内有效
- 需要时重新上传

### 频率限制

- 上传：30 次/分钟，1000 次/小时
- 发送：30 条/分钟，1000 条/小时
